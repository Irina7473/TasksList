Программа для создания списка задач.

Запустите файл TasksListWpfApp.exe

Кнопка Создание нового списка задач обнуляет существующий список и создает новый.
Кнопка Добавление новых задач в список позволяет добавить новые задачи и активизировать кнопку Записать задачу.
Кнопка Загрузка списка задач из файла загружает ранее сохраненный в файл список в Готовый список задач.
Кнопка Запись списка задач в текстовый файл сохраняет Готовый список задач в текстовый файл.

Задачи в Готовом списке задач можно удалить или изменить нажатием правой кнопки мыши на соответствующую задачу.
При выборе пункта меню Изменить задача попадает в форму формирования задачи. После изменения нажмите кнопку Изменить задачу.

На закладке Таблица важности по нажатию кнопки Создать свою таблицу важности можно выбрать свои цвета для каждого уровня или одного из уровней.
Кнопка Сохранить выбор запоминает выбранные цвета.
Кнопка Обновить список меняет цвета в заполненном Готовом списке задач.

В строке Состояние отражаются действия в программе.
